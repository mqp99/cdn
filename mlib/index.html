    Get CDN link

    When you only need to include MLib’s compiled CSS or JS, you can use MLibCDN.
    See it in action with our simple starter template, or browse the examples to jumpstart your next project.

    <!-- CSS only -->
    <link rel='stylesheet' href='https://mqp99.github.io/cdn/mlib/css/app.min.css' />

    <!-- Jquery, MLibJS -->
    <script src='https://cdnjs.cloudflare.com/ajax/libs/jquery/3.5.1/jquery.min.js'></script>
    <script src='https://mqp99.github.io/cdn/mlib/js/mlib.js'></script>
